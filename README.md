# ukpolice

Nim wrapper for the UK Police Data API

## Installation

- Run this command in your project directory:

```bash
nimble install ukpolice
```

## Usage

### Import the package

```nim
import ukpolice
```

## Tests

- To run tests, simply run this command:

```nim
nimble test -d:ssl
```
